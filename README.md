# NC News App

A Reddit-inspired article and comment platform where users can browse articles, vote on content, and engage in discussions as a logged-in user.

## 🔗 Live Demo

**Deployed Application:** [https://emerald-news-webservice.netlify.app/](https://emerald-news-webservice.netlify.app/)

## 📦 Backend Repository

**Backend Repo:** [https://github.com/NGalletly/BoroughBooks-BE](https://github.com/NGalletly/BoroughBooks-BE)

The backend API is built with PostgreSQL and hosted on Supabase (database) and Render (server). Please note that the server may take a moment to spin up if it has been idle.

## ⚙️ Minimum Requirements

- **Node.js:** v25.1.0 or higher

## 🚀 Running the Project Locally

### Backend Setup

1. **Clone and set up the backend:**

```bash
   git clone https://github.com/NGalletly/BoroughBooks-BE
```

2. **Follow the instructions in the backend README** to install dependencies, configure environment variables, and start the server.

### Frontend Setup

1. **Clone the frontend repository:**

```bash
   git clone https://github.com/NGalletly/nc-news-FE
```

2. **Navigate to the project directory:**

```bash
   cd nc_news
```

3. **Install dependencies:**

```bash
   npm install
```

4. **Start the development server:**

```bash
   npm run dev
```

Vite will start a local development server and provide you with a local URL.
